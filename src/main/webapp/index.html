About StreamVerse

StreamVerse is a modern movie streaming web interface inspired by Netflix. It provides users with a smooth and interactive experience to browse trending movies, popular shows, and entertainment content in a visually rich layout. This project is built using only HTML, CSS, and JavaScript to demonstrate frontend design skills, responsive layout techniques, and modern UI effects like hover animations, horizontal scrolling rows, and glass-style navigation.
